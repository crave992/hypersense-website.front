Introducing the Be Outdare Project

We created an all-in-one solution for Be Outdare, a sports bike rental startup, building both their back-end database and their front-end app and website.

Rider Experience

An elegant front-end experience makes it easy for riders to create user accounts and even their own customized cycling information feeds.

Boosting Business

A robust back-end database connects riders with cycling stores, generating Be Outdare commission for each successful rental.

Technical Specs

The front-end for the Be Outdare website and native app was developed with cross-platform Angular. The back-end and API were built with Node.js and Loopback.

The Be Outdare database is AWS Aurora MySQL- compatible. Hosting is on AWS in a self-scaling, self-recovery-enabled environment.
