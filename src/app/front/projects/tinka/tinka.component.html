Get your cash instantly

Fintech solution designed for the Dutch market, Tinka allows users to quickly purchase using spread payments for multiple merchants. The users can also loan money through the app. Mobile app developed and shipped with Flutter at the core.

Key features

Features make or break an app. Below you can find some of the features that stand out for this app.

Biometric authentication

Makes use of all device sensors

Flutter integrates with the device biometric authentication methods for both Android and iOS. Data is secure since Flutter gets only a confirmation, not any real data for authentication

Big data handled with ease

Using smart database components

Facilitating lots of transactions means that everything must work flawlessly. With Flutter and it's integrations you can have peace of mind that your users have the best experience possible with almost no delays or impediments.

Integrates with 3rd parties

Multiple ways to get approved fast

Tinka allows shoppers to quickly get approved for a loan which can then be paid directly at merchants
